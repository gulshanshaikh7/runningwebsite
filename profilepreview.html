These are profile previews of how the App Screen's will appear on your mobile device.

Weekly Goal Dashboard

This screen helps users track their running progress with an interactive weekly goal slider showing completed distance. It displays current run stats, including pace and distance, along with a summary of previous running activities like solo runs, group runs, and races. It's a hub for maintaining running consistency.

#Interactive #Progress #Goals

Profile and Personal Stats

The Profile screen showcases the user's photo and their weekly progress under "Total Progress," covering distance, time, and calories burned. It includes tools for setting personal running goals, viewing achievements, and managing app preferences such as dark mode and metrics. A dedicated "Help" section supports user safety and troubleshooting needs.

#Stats #Achievements #Safety

Run Map and Metrics

This screen provides a detailed view of the user's current or last run, featuring an interactive map and key metrics such as distance, run time, calories burned, and speed. It's designed to give users a clear breakdown of their performance for each run. The visual map feature helps user's visualize their next run routes adding creativity and more fun to the exercise.
